📲 Hướng dẫn sử dụng lịch nhắc nhở

Sau khi bấm 📅 Xuất lịch nhắc nhở (.ics), bạn tải về file .ics — định dạng lịch chuẩn quốc tế dùng được trên mọi thiết bị.

💡 Khuyến nghị: Tạo một lịch riêng mang tên "Lịch Giỗ Dòng Họ" thay vì nhập vào lịch mặc định — để sau này có thể bật/tắt hoặc xoá toàn bộ mà không ảnh hưởng đến lịch cá nhân.

🍎 iPhone / iPad (iOS & iPadOS)

📥 File .ics lưu ở đâu? Safari lưu vào Tệp (Files) → Đã tải xuống. Chrome/Firefox lưu vào Tệp → Trên iPhone này → [Tên trình duyệt]. Kéo thanh thông báo xuống để thấy "Đã tải xong".

Cách 1 — Ứng dụng Tệp (dễ nhất):

  1. Mở app Tệp (Files) → thư mục Đã tải xuống → tìm file Lich_gio.ics
  2. Nhấn vào file → iOS hỏi "Thêm tất cả X sự kiện?" → nhấn Thêm
  3. Chọn lịch → Thêm

Cách 2 — Qua email: Gửi file .ics vào email, mở mail → nhấn file đính kèm → Thêm tất cả.

⚠ Safari trên iPhone không mở file .ics trực tiếp — dùng Files hoặc email.

👉 Tạo lịch riêng: Mở app Lịch → nhấn tab Lịch (góc dưới cùng) → Thêm lịch → đặt tên "Lịch Giỗ Dòng Họ" → khi nhập .ics, chọn lịch này.

🤖 Android

Cách 1 — Google Calendar trên máy tính (khuyến nghị):

  1. Mở calendar.google.com trên trình duyệt
  2. ⚙ Cài đặt → Nhập & xuất → Nhập
  3. Chọn file .ics → chọn lịch → Nhập
  4. Lịch tự đồng bộ về điện thoại qua tài khoản Google

📥 File .ics được lưu ở đâu? Chrome Android tự lưu vào Tải xuống. Mở app Quản lý tệp (hoặc File Manager / Tệp của tôi) → Tải xuống → tìm file Lich_gio.ics. Hoặc kéo thanh thông báo xuống → nhấn "Đã tải xong" để mở ngay.

Cách 2 — Trực tiếp trên điện thoại: Google Calendar → ☰ → Cài đặt → Nhập sự kiện → chọn file .ics trong thư mục Tải xuống.

👉 Tạo lịch riêng: Trong Google Lịch → nhấn Lịch khác → Tạo lịch mới → đặt tên "Lịch Giỗ Dòng Họ" → khi nhập .ics chọn lịch này.

📱 Tạo shortcut trên điện thoại / máy tính bảng

Thêm link GitHub Pages vào màn hình chính để mở nhanh như một ứng dụng:

🍎 iPhone / iPad (Safari):

  1. Mở link GitHub Pages bằng Safari
  2. Nhấn biểu tượng Chia sẻ ở thanh dưới màn hình (hình ô vuông có mũi tên hướng lên)
  3. Kéo xuống danh sách → chọn "Thêm vào Màn hình chính"
  4. Đặt tên "Lịch Giỗ Dòng Họ" → nhấn Thêm

🤖 Android (Chrome):

  1. Mở link GitHub Pages bằng Chrome
  2. Nhấn ⋮ (3 chấm dọc, góc trên phải)
  3. Chọn "Thêm vào màn hình chính" hoặc "Cài ứng dụng"
  4. Nhấn Thêm → biểu tượng xuất hiện trên màn hình

🤖 Android (Samsung Internet):

  1. Mở link GitHub Pages
  2. Nhấn ☰ (góc dưới phải) → chọn "Thêm trang vào" → "Màn hình chính"

💡 Sau khi thêm shortcut, ứng dụng mở toàn màn hình như app thật!

📅 Nhắc nhở tự động theo từng loại giỗ:
• Cúng: nhắc trước 15 ngày, 7 ngày và 3 ngày
• Gửi: nhắc trước 3 ngày và 1 ngày
• Nhớ ngày: nhắc trước 1 ngày

File .ics xuất cả năm hiện tại lẫn năm sau để không bỏ lỡ ngày gần nhất.
