📂 Khôi phục dữ liệu

Chọn file .json đã sao lưu trước đó. Dữ liệu hiện tại sẽ được thay thế hoàn toàn bằng dữ liệu từ file.

💾 Lưu ý về dữ liệu

Toàn bộ dữ liệu ngày giỗ được lưu trực tiếp trên thiết bị của bạn (bộ nhớ trình duyệt - localStorage), không gửi lên bất kỳ máy chủ nào.

Điều này có nghĩa là:

  • 📵 Không đồng bộ giữa các thiết bị - dữ liệu nhập trên điện thoại sẽ không hiện trên máy tính và ngược lại
  • 🔒 Hoàn toàn riêng tư - không ai khác có thể xem dữ liệu của bạn
  • ⚠️ Xoá dữ liệu trình duyệt/đổi trình duyệt sẽ làm mất toàn bộ dữ liệu đã nhập

👉 Khuyến nghị: Khi thêm dữ liệu mới, hãy dùng chức năng Sao lưu dữ liệu và gửi mail cho chính mình file đã tạo để lưu trữ - hoặc dùng cách khác thuận tiện cho bạn như lưu vào Drive, iCloud...

Tạo shortcut trên điện thoại / máy tính bảng

Thêm vào màn hình chính để mở nhanh như một ứng dụng thật

🔗 Link ứng dụng: https://sumo2025.github.io/lich-gio

🍎 iPhone / iPad (Safari)

  1. Mở link bằng Safari (bắt buộc — Chrome trên iOS không hỗ trợ)
  2. Nhấn biểu tượng Chia sẻ ở thanh dưới màn hình (hình ô vuông có mũi tên hướng lên)
  3. Kéo xuống danh sách → chọn "Thêm vào Màn hình chính"
  4. Đặt tên "Lịch Giỗ Dòng Họ" → nhấn Thêm

🤖 Android — Chrome

  1. Mở link bằng Chrome
  2. Nhấn ⋮ (3 chấm dọc, góc trên phải)
  3. Chọn "Thêm vào màn hình chính" hoặc "Cài ứng dụng"
  4. Nhấn Thêm → biểu tượng xuất hiện ngay trên màn hình

🤖 Android — Samsung Internet

  1. Mở link bằng Samsung Internet
  2. Nhấn ☰ (góc dưới phải) → chọn "Thêm trang vào" → "Màn hình chính"
✅ Sau khi thêm shortcut, ứng dụng mở toàn màn hình như app thật — không cần mở trình duyệt, không cần nhớ link!

Hướng dẫn sử dụng file lịch nhắc nhở (.ics)

Nhập vào Google Lịch / Apple Lịch để nhận nhắc nhở tự động trước ngày giỗ

💡 Khuyến nghị: Tạo lịch riêng tên "Lịch Giỗ Dòng Họ" — để sau này tắt/xóa riêng mà không ảnh hưởng lịch cá nhân.

🍎 iPhone / iPad (iOS & iPadOS)

📥 File lưu ở đâu? Safari → Tệp → Đã tải xuống. Chrome/Firefox → Tệp → Trên iPhone này → [Tên trình duyệt].

Cách 1 — Ứng dụng Tệp (dễ nhất)

  1. Mở app Tệp → Đã tải xuống → tìm file Lich_gio.ics
  2. Nhấn vào file → chọn lịch → nhấn Thêm tất cả

Cách 2 — Qua email: Gửi file về email → mở đính kèm → nhấn Thêm tất cả.

⚠ Safari trên iPhone không mở .ics trực tiếp — phải dùng Tệp hoặc email.

👉 Tạo lịch riêng: App Lịch → tab Lịch (dưới cùng) → Thêm lịch → đặt tên → khi nhập .ics chọn lịch đó.

🤖 Android

📥 File lưu ở đâu? Chrome tự lưu vào Tải xuống. Mở Quản lý tệp → Tải xuống → tìm Lich_gio.ics. Hoặc kéo thanh thông báo → nhấn "Đã tải xong" để mở ngay.

Cách 1 — Google Calendar trên máy tính (khuyến nghị)

  1. Mở calendar.google.com trên máy tính
  2. ⚙ Cài đặt → Nhập & xuất → Nhập
  3. Chọn file Lich_gio.ics → chọn lịch → nhấn Nhập
  4. Lịch tự đồng bộ về điện thoại qua tài khoản Google

Cách 2 — Trực tiếp trên điện thoại: Google Lịch → ☰ → Cài đặt → Nhập sự kiện → chọn file trong thư mục Tải xuống.

👉 Tạo lịch riêng: Google Lịch → Lịch khác → Tạo lịch mới → đặt tên → khi nhập .ics chọn lịch đó.
