Các thao tác trong mục Sửa dữ liệu

  • ✏️ Chỉnh sửa tên, ngày tháng, loại giỗ, ghi chú từng mục
  • 💾 Sao lưu toàn bộ dữ liệu ra file JSON
  • 📂 Khôi phục dữ liệu từ file sao lưu
  • 🗑️ Xóa toàn bộ dữ liệu (có xác nhận trước khi thực hiện)
⚠️ Ứng dụng chỉ nhận đúng 3 loại giỗ: cung (cúng tại nhà) - gui (gửi lễ đi nơi khác) - nho (chỉ ghi nhớ). Nhập sai thì bản ghi bị bỏ qua.

Cách 1 - Từ ảnh chụp / scan / file văn bản

Mở ChatGPT, Claude (claude.ai) hoặc Gemini - đính kèm ảnh hoặc dán nội dung - gửi kèm đoạn nhắn tin sau:

"Hãy chuyển danh sách ngày giỗ này thành file JSON. Mỗi mục gồm các trường: id (số thứ tự bắt đầu từ 1), name (tên người được giỗ), month (tháng âm lịch, số từ 1 đến 12), day (ngày âm lịch, số từ 1 đến 30), cat (loại giỗ - chỉ được dùng đúng một trong 3 từ: cung, gui, hoặc nho), note (ghi chú ngắn hoặc để trống "").
Cách tôi phân loại: [ví dụ: "giỗ chính" thì dùng cung, "giỗ gửi" thì dùng gui, còn lại dùng nho]"

Lưu file JSON (Windows)

  1. 1Mở Notepad - dán nội dung JSON vào
  2. 2File - Save As - ô "Save as type" chọn All Files
  3. 3Đặt tên lichgio.json - nhấn Save

Cách 2 - Từ file Excel

Tải file Excel mẫu lichgio_mau_nhap_du_lieu.xlsx - điền vào - chụp ảnh hoặc copy - nhờ AI chuyển sang JSON như Cách 1.

Nhập file JSON vào ứng dụng

  1. 1Nhấn Cài đặt (góc trên phải)
  2. 2Chọn Khôi phục dữ liệu
  3. 3Chọn file lichgio.json - xác nhận
⚠️ Khôi phục sẽ thay thế toàn bộ dữ liệu hiện tại. Hãy Sao lưu trước nếu đang có dữ liệu cũ.
  1. 1Trên máy tính: Cài đặt - Sao lưu dữ liệu - lưu file JSON về máy
  2. 2Gửi file sang điện thoại qua Zalo (nhắn cho chính mình), email, hoặc Google Drive / iCloud
  3. 3Trên điện thoại - lưu file xuống bộ nhớ:
    Zalo: nhấn giữ file - Lưu về máy | Email: nhấn file - Tải xuống
  4. 4Mở ứng dụng - Cài đặt - Khôi phục dữ liệu - chọn file - xác nhận
👨‍👩‍👧 Chia sẻ cho người thân: gửi link ứng dụng kèm file sao lưu - họ thực hiện từ Bước 3 là có ngay toàn bộ danh sách ngày giỗ.
